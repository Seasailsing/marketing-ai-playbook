> ## Prompt Library (Work in Progress)
>
> A set of prompts I've found consistently useful in a GTM context:
>
> **Strategic positioning brief**
> ```
> You are a B2B GTM strategist. Given the following company description and target customer, write a one-paragraph strategic positioning statement that: (1) names the specific problem solved, (2) identifies who it's for, (3) articulates the unique approach, and (4) includes a proof point.
> ```
>
> **Weekly pipeline summary**
> ```
> Summarise the following CRM pipeline data into a 5-bullet executive update. Flag deals at risk, highlight top movers, and recommend one action for this week.
> ```
>
> **Localisation review**
> ```
> Review the following marketing copy for cultural fit in [market]. Flag any phrases, references, or tones that may not land well, and suggest alternatives.
> ```
